##  完成Boss登录页面

`src/Renligou.Boss`项目类型为`Asp.net Core Blazor Web`， 渲染模型为Server。 

项目引入了`Ant Design Blazor`作为基础UI框架。

定制主题位置`src/Renligou.Boss/wwwroot/app.css`,产生新的CSS也配置在这里。

登录页已建立 `src/Renligou.Boss/Components/Pages/Login.razor`, 目前为空，需要你完成：

- 左右两栏设置， 比例为6:4

- 左侧背景为主色， 放置站点名称"人力狗"，以及一张贴合人力资源工具的展示图，图由你提供， 放在`src/Renligou.Boss/wwwroot/images`里。

- 右侧内容为登录表单，包含用户名、密码、记住我、忘记密码、注册账号等选项， 要有TAB切换账号登录和手机验证码登录

要求
- 专业、美观， 把自己带入最伟大网页设计师，全人类的设计经验你都掌握。
- 响应式布局， 最小屏幕800*600px
